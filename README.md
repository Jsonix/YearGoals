# Year Goals - Веб-приложение для управления годовыми целями

Простое веб-приложение для отслеживания и управления вашими годовыми целями.

## Структура проекта

```
YearGoals/
├── index.html      # Главная страница
├── css/
│   └── style.css   # Стили приложения
├── js/
│   └── main.js     # Логика приложения
└── README.md       # Документация
```

## Функциональность

- ✅ Добавление новых целей
- ✅ Отметка целей как выполненных
- ✅ Удаление целей
- ✅ Сохранение данных в localStorage браузера
- ✅ Современный и красивый интерфейс

## Как запустить

1. Откройте файл `index.html` в браузере
2. Или используйте Live Server в IntelliJ IDEA:
   - Правый клик на `index.html` → Open in Browser
   - Или используйте встроенный веб-сервер

## Технологии

- HTML5
- CSS3 (с градиентами и анимациями)
- Vanilla JavaScript (без фреймворков)
