# Year Goals - Веб-приложение для управления годовыми целями

Простое веб-приложение для отслеживания и управления вашими годовыми целями.

## Структура проекта

```
YearGoals/
├── index.html      # Главная страница
├── css/
│   └── style.css   # Стили приложения
├── js/
│   └── main.js     # Логика приложения
└── README.md       # Документация
```

## Функциональность

- ✅ Добавление новых целей
- ✅ Отметка целей как выполненных
- ✅ Удаление целей
- ✅ Сохранение данных в localStorage браузера
- ✅ Современный и красивый интерфейс

## Как запустить локально

### Вариант 1: Простой запуск (без сервера)
Просто откройте файл `index.html` в браузере:
- Двойной клик на `index.html`
- Или перетащите файл в окно браузера

### Вариант 2: Локальный HTTP-сервер (рекомендуется)

#### Используя Python (уже установлен):
```bash
# Запустить сервер на порту 8000
python3 -m http.server 8000

# Или использовать скрипт
./start-server.sh
```

После запуска откройте в браузере: `http://localhost:8000`

#### Используя Node.js (если установлен):
```bash
npx http-server -p 8000
```

### Вариант 3: IntelliJ IDEA Live Server
- Правый клик на `index.html` → Open in Browser
- Или используйте встроенный веб-сервер

## Как перезапустить

1. **Остановить сервер**: Нажмите `Ctrl+C` в терминале
2. **Запустить снова**: Выполните команду запуска заново
3. **Обновить страницу в браузере**: Нажмите `F5` или `Cmd+R` (Mac) / `Ctrl+R` (Windows/Linux)

### Если порт занят (ошибка "Address already in use"):

**Вариант 1: Автоматический поиск свободного порта** (рекомендуется)
```bash
./start-server.sh
```
Скрипт автоматически найдет свободный порт, если 8000 занят.

**Вариант 2: Остановить процесс, занимающий порт**
```bash
# Найти процесс на порту 8000
lsof -ti:8000

# Остановить процесс (замените PID на номер процесса)
kill <PID>

# Или одной командой
kill $(lsof -ti:8000)
```

**Вариант 3: Использовать другой порт**
```bash
python3 -m http.server 8001
# или
./start-server.sh 8001
```

## Как смотреть логи

### В браузере (основной способ):
1. Откройте **DevTools** (Инструменты разработчика):
   - `F12` или `Cmd+Option+I` (Mac) / `Ctrl+Shift+I` (Windows/Linux)
   - Или правый клик → "Просмотреть код" / "Inspect"
2. Перейдите на вкладку **Console** (Консоль)
3. Здесь вы увидите:
   - `console.log()` сообщения
   - Ошибки JavaScript
   - Предупреждения
   - Информацию о загрузке ресурсов

### Полезные команды в консоли:
```javascript
// Очистить localStorage (сбросить все сохраненные цели)
localStorage.clear();

// Посмотреть сохраненные цели
JSON.parse(localStorage.getItem('year2026Goals'));

// Очистить консоль
console.clear();
```

### В терминале (если используете сервер):
Логи сервера будут отображаться в терминале, где запущен сервер (запросы к файлам, ошибки 404 и т.д.)
